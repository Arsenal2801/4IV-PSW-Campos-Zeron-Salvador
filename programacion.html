PROGRAMACION

En esta seccion te estare hablando de los lenguajes que mas o menos conozco ademas de algunos de los proyectos que estoy llevando a cabo.

  1. C:

    Este lenguaje lo aprendi en 3° de secundaria en un curso que estuve tomando con mi papá durante el cual estuvimos haciendo los proyectos con el microprocesador llamado "Arduino" este fue el punto en el que me empezo a llamar mucho la atencion el area de la programacion y al tener una sintaxis muy simple me fue muy facil aprenderlo y dominar este lenguaje a un 50%

  2. Java

    Este lo aprendi al entrar a la vocacional donde estoy cursando la carrera tecnica de programacion.
    Con este lenguaje se desarrollo la aplicacion de "Calendar Notification Plus"("CaNot Plus") que como su nombre lo indica es una aplicacion que tiene la funcion de un calendari de actividades para que el usuario registre sus tareas y le mande una notificacion antes de que esta caduque. Actualmente este proyecto se esta trabajando en 2° plano es decir no lo estoy trabajando como un proyecto serio sino que estoy utilizandolo para seguir aprendindo de este lenguaje e incluso si queda funcional podria incluso publicarla

  3. Java Script:

    Actualmente tengo conocimiento realmente muy basico de este lenjuage realmente desearia aprenderlo todo pues me parece que este lenguaje multiparadigma es muy funcional ya que basicamente es utilizado en todas partes ya que una gran parte de sitios web y aplicaciones como Paypal, Netflix, LinkedIn y Uber lo utilizan de una u otra forma.

  4. Python:

    Python es un lenguaje de programación interpretado cuya filosofía hace hincapié en la legibilidad de su código. Podríamos decir que es un lenguaje de programación multiparadigma orientado a objetos, programación imperativa y programación funcional. Es un lenguaje interpretado, dinámico y multiplataforma.
    En este lenguaje tengo un proyecto para crear una IA que me ayude a controlar mi casa como la famosa inteigencia artificial "Jarvis" de Iron Man este es el objetivo aunque no le dedico mucho tiempo a este proyecto pues se que para crear una IA es un proceso largo entonces le dedico 1 hora al dia y llevo unas 2 semanas con este proyecto.
